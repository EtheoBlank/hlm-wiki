## 3. 引用格式

### 3.1 正文引用格式

**程乙本正文**：

```
> 《红楼梦》第三回"贾雨村夤缘复旧职 林黛玉抛父进京都" · 程乙本 · 
> 俞平伯校注 · 人民文学出版社 · 1982 · 页 38
```

**脂本异文**：

```
> [异文] 庚辰本此处作"……"（俞平伯辑《脂砚斋红楼梦辑评》页 124）
```

**脂批**：

```
> [脂批] 甲戌本第三回侧批："……"
> [脂批] 庚辰本第二十一回回前批："……"
```

**诗词判词**：

```
> [判词] 第一回 · 程乙本 · 页 9
> "……"（正文）
> Hawkes 译文：……（Vol. 1, p. 67）
```

### 3.2 英译引用格式

**Hawkes**：

```
> Hawkes, *The Story of the Stone*, Vol. 1: *The Golden Days*, 
> Penguin Classics, 1973, p. 67.
```

**Minford**：

```
> John Minford, in Hawkes & Minford, *The Story of the Stone*, 
> Vol. 4: *The Debt of Tears*, Penguin, 1982, p. 210.
```

### 3.3 学术引用格式

```
> 欧丽娟，《大观红楼》，北京大学出版社，2018，页 245。
> C. T. Hsia, *The Classic Chinese Novel*, Columbia University Press, 1968, p. 245.
```

## 4. 引用页脚模板

每页底部格式：

```
---
**版本说明**：
- 正文以俞平伯校注本（人文社 1982）为底
- 脂批异文参照俞平伯辑《脂砚斋红楼梦辑评》（人文社 1975）与冯其庸校批本
- 英译以 Hawkes《The Story of the Stone》（Penguin 1973–1986）为底
- 存疑之处以 [待校] 标注

最后更新：YYYY-MM-DD
```

## 5. 资料源优先级规则

遇到版本冲突时，按以下优先级：

1. **情节冲突**（同一事件不同描写）：以程乙本为主，脂本异文列入脚注
2. **脂批解读**（同一批语不同理解）：以最早版本（甲戌本）为优，庚辰本次之
3. **人物结局**（后四十回 vs 脂批推测）：**两栏并列**，明确标注"程乙本结局"与"脂批推测结局"
4. **年龄、时间数字**（不同版本差异大）：以程乙本为基，异文标注
5. **诗词版本**（判词等）：以脂本系统为优（早于程乙本）
