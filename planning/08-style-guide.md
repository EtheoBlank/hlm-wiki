# 08 · 文笔与翻译规范

> **目标**：保证 400+ 人物的中英文写作风格一致、文笔"信达雅"。  
> **核心原则**：中文以"程乙本为底 + 现代白话译注双栏"；英文以 Hawkes 风格为参照 + 本项目精修。

---

## 1. 现代白话译注规则

### 1.1 三栏对照

每段原文三栏：

```
[原文]     《红楼梦》第 X 回 · 程乙本 · 页 X
[今译]     现代白话散文意译
[Hawkes]   Penguin 1973, p. X（英译参照）
[译注]     异文、典故、双关、版本异文（可选）
```

### 1.2 现代白话译注规范

| 规则 | 示例 |
|------|------|
| **散文体**：不用半文半白 | ❌"黛玉不胜其悲" → ✅"黛玉禁不住悲伤起来" |
| **直译为主，意译为辅**：忠于原意 | ✅（先直译，再润色） |
| **保留人物语气**：黛玉尖刻、宝玉痴、凤姐伶俐 | 详见各人物"语气" |
| **现代汉语规范**：不用"咱们"、"咱们家"等口语化 | ✅"我们" |
| **专名不译**：人名、地名保留原字 | "林黛玉"、"金陵" |
| **语气词保留**："罢"、"呢"、"啊" | ✅"怎么不去呢" |

### 1.3 文气

- **判词**：典雅工整；用现代文言 + 注释双栏
- **诗词**：现代白话诗译，押韵（不强求）；保留原诗意境
- **诔文**：典雅；可仿古
- **对话**：口语化；符合人物性格
- **叙事**：中等文雅；接近现代散文

## 2. 英文翻译规则

### 2.1 风格参照

| 维度 | 参照 |
|------|------|
| **底本** | Hawkes《The Story of the Stone》 |
| **风格** | 现代英语、明晰流畅、文笔优雅 |
| **专名** | 见 `07-glossary.md` |
| **人名** | 拼音（无连字符、无声调） |
| **专名首次出现** | 加拼音 + 关系说明 |

### 2.2 信达雅

| 维度 | 实施 |
|------|------|
| **信（Faithfulness）** | 忠于程乙本；Hawkes 偏离处脚注标出 |
| **达（Expressiveness）** | 现代英语流畅；避免 18 世纪英译腔 |
| **雅（Elegance）** | 诗、判词、诔文统一重译；术语一致 |

### 2.3 不译项（保留原字 + 脚注）

- **判词双关**："玉带林中挂"（黛玉）/ "金簪雪里埋"（宝钗）→ 保留拼音
- **人名双关**：
  - "甄士隐"（真事隐）
  - "贾雨村"（假语村）
  - "林黛玉"（代玉）/ "薛宝钗"（钗）/ "贾元春"（元日）
  - "王熙凤"（稀凤 → 凡鸟）→ 脚注说明"凡鸟偏从末世来"
- **神话术语**："绛珠仙草"、"神瑛侍者"、"通灵宝玉" → 拼音或保留
- **大观园地名**：保留汉语意译 + 拼音

### 2.4 Hawkes 风格继承

| Hawkes 译法 | 本项目继承 |
|------------|----------|
| "The Story of the Stone" 整体书名 | ✅ 继承 |
| "The Golden Days" Vol. 1 副标题 | ✅ 继承 |
| "Jia Bao-yu" → "Jia Baoyu"（去连字符） | ⚠️ 微调 |
| "Lin Dai-yu" → "Lin Daiyu" | ⚠️ 微调 |
| "Happy Red Court"（怡红院） | ✅ 继承 |
| "Naiad's House"（潇湘馆） | ❌ 改 "Bamboo Lodge" |
| "Cold Fragrance Pills" | ✅ 继承 |
| "Precious Jade of Spiritual Understanding" | ✅ 继承 |
| "Disenchantment"（警幻仙姑） | ✅ 继承 |
| "Won-Done Song"（好了歌） | ✅ 继承 |
| "Crimson Pearl Flower"（绛珠仙草） | ✅ 继承 |

### 2.5 不采用 Hawkes 之处

| Hawkes 译法 | 替代 |
|------------|------|
| "Naiad's House"（潇湘馆） | "Bamboo Lodge" |
| "The Land of Illusion"（太虚幻境） | 保留 "The Land of Illusion" + 拼音注 |
| "The Office of…Frailty"（薄命司等） | "The Office of Frail Lives" |
| "Bondservant"（丫鬟/奴才） | "Maid" / "Servant" |
| 简化典故（如把"谢道韫咏雪"译为雪景比喻） | 保留典故 + 脚注 |

## 3. 写作禁忌

| ❌ 禁忌 | ✅ 应改为 |
|--------|----------|
| 添加原著没有的情节 | 严格忠于原文 |
| 现代价值观评断人物 | 仅作红学观点引用 |
| 87 版电视剧人物形象描述 | 以原文/脂批描述为准 |
| 现代词汇（"宅斗""CP"） | 用红学界通用术语 |
| 简繁混用 | 全站用简体中文；术语首次出现加繁体 |
| 网络流行语 | 用规范的现代汉语 |
| 自创判词解读 | 引用红学家观点 |

## 5. 标点与格式

### 5.1 中文标点

- 用全角中文标点：，。：；！？""''（）《》
- 引号："  "
- 书名号：书名《红楼梦》/ 西厢记
- 引文：

```
> 黛玉笑道："我来的不巧了。"
```

### 5.2 英文标点

- 用半角英文标点：, . : ; ! ? " ' () —
- 引文：
  > "I've been there before," said Dai-yu.

### 5.3 中英混排

- 中文段落中夹英文：英文前后留空格
- 例：黛玉住在 Bamboo Lodge（潇湘馆）

## 6. 引用规范

### 6.1 章节出处

格式：

```
《红楼梦》第 X 回 · 程乙本 · 俞平伯校注 · 人民文学出版社 · 1982 · 页 X
```

### 6.2 脂批

```
[脂批·甲戌本] 第 X 回侧批
[脂批·庚辰本] 第 X 回双行夹批
[脂批·戚序本] 第 X 回回前批
```

### 6.3 Hawkes

```
Hawkes, *The Story of the Stone*, Vol. X: *Title*, Penguin Classics, 1973–1986, p. X.
```

### 6.4 学术引用

```
欧丽娟，《大观红楼》，北京大学出版社，2018，页 X。
Hawkes, "Introduction", *The Story of the Stone*, Vol. 1, p. X.
```

## 7. [待校] 风格存疑

- [待校] 现代白话译注保留多少古语词（如"罢了"、"呢"、"也罢"）
- [待校] 是否采用"信""雅"并重（保留原句式 vs 流畅英文）
- [待校] 诗词翻译是否要求押韵
- [待校] 是否给每段原文加"今译"还是只对关键段加
- [待校] 英译是否需加 IPA（国际音标）
