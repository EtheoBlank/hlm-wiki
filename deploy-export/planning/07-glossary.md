## 10. 翻译风格规则

| 原则 | 实施 |
|------|------|
| **人名拼音** | 全部使用 Pinyin（无连字符、无声调） |
| **地名拼音** | 全部使用 Pinyin |
| **称呼** | 关系称谓保留汉语意译（Lady Wang、Grandmother Jia），加拼音注释 |
| **官职** | 意译 + 拼音首注 |
| **典故** | 直译 + 脚注解释 |
| **诗词** | 押韵英文翻译；优先用 Hawkes 风格；保留原诗主题 |
| **人名双关** | "甄士隐" = "真事隐" → 保留原字 + 脚注 |
| **"黛"谐音"代"** | 保留原字；脚注说明"代玉"暗示 |

## [待校] 术语存疑

- [待校] "怡红院" Hawkes 用 "Happy Red Court"，本项目沿用；可考虑 "House of Red Delights"
- [待校] "潇湘馆" Hawkes 用 "Naiad's House"（水妖居所），红学界有争议；本项目用 "Bamboo Lodge"
- [待校] "蘅芜苑" "Alpinia Park" 由 "蘅芜"（一种香草）音译；需统一
- [待校] "栊翠庵" "Green Lattice Temple" 是否合适；"栊"指窗棂
