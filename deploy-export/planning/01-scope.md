# 01 · 项目范围与决策

## 1. 范围

| 维度 | 决策 |
|------|------|
| 人物覆盖 | 红楼梦全书有名有姓人物（约 400+ 人）；按 Tier A（核心 18）/ B（重要配角 80+）/ C（次要人物 300+）分级 |
| 文本范围 | 程乙本 120 回完整覆盖；脂批作为八十回后结局的推测依据 |
| 章节范围 | 楔子（甄士隐梦幻识通灵）+ 第一回至第一百二十回 |
| 时间锚点 | 以书中纪年为主，参照脂批提示的历史背景（明末清初） |
| 地理范围 | 京都、扬州、金陵、苏州、长安、大同 等书中出现的地名 |

## 2. 底本与版本策略

### 2.1 主底本（正文引用以这一版为准）

- **程乙本 · 俞平伯校注 · 人民文学出版社（1982 年初版）**
  - 红学界通行权威；章回完整（120 回）；后四十回亦收录
  - 校注严谨，正文 + 校记 + 注释三栏
  - 引用时标注：《红楼梦》[回目] · 程乙本 · 人文社俞校本（1982），页 X

### 2.2 异文底本（脚注形式补充）

| 版本 | 别称 | 特点 | 用法 |
|------|------|------|------|
| **庚辰本** | 脂庚本、《脂砚斋重评石头记》庚辰秋月定本 | 曹雪芹生前最后定本之一；存 78 回（缺 64、67 回） | 主要异文参照 |
| **甲戌本** | 脂甲本、石头记甲戌残本 | 最早脂本（1754），仅存 16 回；有大量脂批 | 楔子与前五回校勘 |
| **戚序本** | 有正本、戚沪本、戚宁本 | 80 回脂本系统 | 高鹗续书前的另一传本 |

| **舒序本** | 舒元炜序本 | 40 回残本 | 校勘 |
| **列藏本** | 列宁格勒藏本 | 78 回脂本 | 异文 |
| **梦稿本** | 杨藏本、梦觉主人序本 | 80 回 | 异文 |

### 2.3 现代权威校注（考据与赏析参照）

- **俞平伯**：《红楼梦研究》《红楼梦简论》《唐六如与林黛玉》
- **周汝昌**：《石头记会真》（2005）、《红楼梦新证》（1953）
- **冯其庸**：《瓜饭楼重校评批红楼梦》（2017）
- **蔡义江**：《红楼梦诗词曲赋全编》《红楼梦校注》
- **欧丽娟**：台大公开课《红楼梦》讲义（2016–）
- **白先勇**：台湾大学《红楼梦》导读（2014）

### 2.4 脂批引用规范

脂批分三类（红学界通用）：

1. **甲戌眉批 / 夹批 / 侧批**——最早、最权威，标注"甲戌本·[位置]"
2. **庚辰眉批 / 双行夹批**——批语最多，标注"庚辰本·[位置]"
3. **蒙府、戚序、靖藏本批语**——孤本或回末总评

脂批如涉及八十回后结局（如"赦老死"、"探春远嫁"、"惜春为尼"、"凤姐扫雪"等），需明确标注"[脂批·推测]"。

## 3. 翻译策略

### 3.1 主底本（英译引用以这一版为准）

- **David Hawkes, *The Story of the Stone*, Penguin Classics, 1973–1986**
  - Vol. 1 *The Golden Days*（1–25 回），1973
  - Vol. 2 *The Crab-Flower Club*（26–53 回），1977
  - Vol. 3 *The Warning Voice*（54–79 回），1980
  - Vol. 4 *The Debt of Tears*（80–98 回，John Minford 续译），1982
  - Vol. 5 *The Dreamer Wakes*（99–120 回，John Minford 译），1986

### 3.2 精修原则

| 原则 | 实施 |
|------|------|
| **信** | 忠于程乙本正文；Hawkes 偏离处用脚注标出 |
| **达** | 现代英语流畅；避免 18 世纪英译腔 |
| **雅** | 诗、判词、诔文、偈语统一重译，Hawkes 风格为参照 |
| **专名** | 见 `07-glossary.md`；人名、地名、佛道术语、官职、礼仪术语全站统一 |
| **异译** | Joly、McHugh、West & Idema 译本可作回退参照 |
| **不可译项** | 红楼梦独有双关（如"甄（真）贾（假）"、"黛（代）玉"）保留原字 + 脚注 |

### 3.3 译注双栏

每段原文三栏呈现：

```
[原文]     《红楼梦》第三回 · 程乙本    [页 X]
[今译]     现代白话散文意译
[Hawkes]   Penguin 1973, p. Y
[译注]     异文、典故、双关、版本异文
```

## 4. 版权与引用

### 4.1 文本版权

- **程乙本原文**：曹雪芹卒于 1763（清乾隆二十八年），距今 260+ 年，已过中国（50 年）与海外各国（多为 70 年）版权保护期 → **可全文引用**
- **脂批**：脂砚斋、畸笏叟批语均为清人原作，已入公有领域 → **可全文引用**
- **程伟元、高鹈序言**：清人作品 → **可全文引用**
- **现代校注**：俞平伯（1900–1990）、周汝昌（1918–2012）、冯其庸（1924–2017）等人的校注文字，已故 30 年以上（按中国著作权法）→ **可作学术引用，标注校注者**
- **Hawkes 译文**：1973–1986 年出版，**仍在版权期内** → **仅作术语参照与脚注引用，不全文摘录**；本项目自行重译

### 4.2 引用格式

```
> 《红楼梦》[回目名称] · [版本] · [校注者] · [出版社] · [年份] · [页码]
> [Hawkes, *The Story of the Stone*, Vol. X · Penguin · 年份 · 页码]
```

## 5. 范围之外（不做的）

- ❌ 后世续书（《红楼梦后梦》《红楼梦补》等）：另开"续书考"子页面
- ❌ 87 版电视剧演员形象图：仅作考据材料，不作人物肖像
- ❌ 红学考据论文争议本身：人物页面只引用主流观点，争议列入"考据脚注"
- ❌ 现存红楼梦版本（甲戌本等）的物理形态考据：属版本学范畴
- ❌ 红学史人物（俞平伯、胡适、周汝昌等）：不属于"红楼梦人物"
