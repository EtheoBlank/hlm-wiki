# Tier C Stub 模板

> 用于创建 Tier C 中未完成人物的最小骨架 profile。

## 用法

1. 复制本文件
2. 修改 `<项目>`、`<角色>` 等占位符
3. 标 [待校] 表示信息不完整
4. 保存到 `content/<家族>/<人物>/profile.md`

## 模板

```markdown
# <角色> / <Pinyin>

> <别称>（如有）

| 项目 | 内容 |
|------|------|
| 别号 | |
| 身份 | |
| 出处 | <回目> |
| 结局 | |

## [待校] 待补全信息

- 详细判词/曲（如适用）
- 性格特征
- 关键事件清单
- 与其他人物关系
- 现代红学家评点

最后更新：YYYY-MM-DD
```
